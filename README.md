
# No Hotbar Looping

Ever wanted your hotbar to not loop when you scroll at the end of it? This mod fixes this!
Simply add this mod to your game, and you can no longer loop through your hotbar.

## Features

- Toasts
- Keybindings
- ModMenu integration
